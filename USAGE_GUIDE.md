# A股行情数据MCP服务 - 使用指导

## 快速开始

### 1. 安装依赖

```bash
# 激活虚拟环境
source .venv/bin/activate

# 安装mcp依赖
uv pip install -e ".[dev]"

# 单独安装mootdx（因为与mcp有httpx版本冲突）
uv pip install mootdx --no-deps
uv pip install pandas
```

### 2. 启动服务

```bash
# 方式1：直接运行启动脚本
python start_stock_market_service.py

# 方式2：使用模块方式运行
python -m src.mcp_servers.stock_market.main
```

### 3. MCP客户端配置

将以下配置添加到MCP客户端（如Claude Desktop）：

```json
{
  "mcpServers": {
    "stock-market-service": {
      "command": "python",
      "args": ["start_stock_market_service.py"],
      "env": {}
    }
  }
}
```

## 功能说明

### 1. 获取1分钟K线数据

**用途：** 获取单日的K线形状和量能情况

**调用示例：**
```
获取贵州茅台(600519)的1分钟K线数据，240条
```

**返回数据包含：**
- `datetime`: 时间戳（如 "2026-06-03 09:30:00"）
- `open`: 开盘价
- `high`: 最高价
- `low`: 最低价
- `close`: 收盘价
- `volume`: 成交量（手）
- `amount`: 成交额（元）

**分析要点：**
- 通过K线形状判断多空力量
- 通过量能变化判断资金进出
- 识别分时支撑/压力位

### 2. 获取日K线数据

**用途：** 获取K线形状、图形、均线压力/支撑位置、量能情况

**调用示例：**
```
获取贵州茅台(600519)的日K线数据，120条
```

**返回数据包含：**

| 字段 | 说明 | 用途 |
|------|------|------|
| `datetime` | 日期 | 时间参考 |
| `open/high/low/close` | OHLC价格 | K线形状 |
| `volume` | 成交量 | 量能分析 |
| `amount` | 成交额 | 资金规模 |
| `ma5` | 5日均线 | 短期趋势/支撑 |
| `ma10` | 10日均线 | 中短期趋势/支撑 |
| `ma20` | 20日均线 | 中期趋势/支撑 |
| `pre_close` | 昨日收盘价 | 计算涨跌幅基准 |
| `limit_up` | 涨停价 | 当日最高限价 |
| `limit_down` | 跌停价 | 当日最低限价 |

**分析要点：**
- **均线系统**：ma5上穿ma10为金叉（看涨），ma5下穿ma10为死叉（看跌）
- **支撑位**：股价回落到均线附近可能获得支撑
- **压力位**：股价上涨到均线附近可能遇到压力
- **涨跌停**：判断股价是否接近涨停/跌停

### 3. 获取实时行情

**用途：** 获取股票当前实时行情

**调用示例：**
```
获取贵州茅台(600519)的实时行情
```

## 数据说明

### 前复权

所有K线数据均已进行**前复权（qfq）**处理：
- 以最新价格为基准，向前调整历史价格
- 保证K线图的连续性
- 消除除权除息对技术分析的影响

### 涨跌停计算

当前使用**主板规则**计算涨跌停：
- 涨停价 = 昨日收盘价 × 1.10
- 跌停价 = 昨日收盘价 × 0.90

**注意：**
- ST股涨跌停为±5%
- 科创板/创业板涨跌停为±20%
- 当前实现使用统一的±10%规则

### 关于流通市值

mootdx的实时行情接口**不直接提供流通市值**数据。如需流通市值，可以通过以下方式获取：

1. **腾讯财经API**：直接返回流通市值（索引[44]）
2. **自行计算**：流通市值 = 当前价格 × 流通股本

## 常见问题

### Q1: 服务启动失败

**问题：** `ModuleNotFoundError: No module named 'mootdx'`

**解决：** 确保已安装mootdx：
```bash
uv pip install mootdx --no-deps
```

### Q2: 获取数据为空

**问题：** 返回 `{"error": "未获取到数据", "data": []}`

**可能原因：**
1. 股票代码格式错误（应为6位数字，如"600519"）
2. 非交易时间获取分钟数据
3. 网络连接问题

**解决：**
- 检查股票代码是否正确
- 日K线数据不受交易时间限制
- 运行 `python -m mootdx bestip -vv` 测试服务器连接

### Q3: 前复权失败

**问题：** 数据未进行前复权处理

**可能原因：**
- 股票无除权除息记录
- 除权除息数据获取失败

**解决：** 系统会自动返回原始数据，不影响使用

|------|---------|------|
| 上海主板 | 6位数字 | 600519 |
| 深圳主板 | 6位数字 | 000001 |
| 创业板 | 6位数字 | 300750 |
| 科创板 | 6位数字 | 688001 |

**注意：** 不需要添加市场前缀（如sh/sz），直接使用6位数字代码即可。
